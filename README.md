# SellerSense

Find hidden gems in classified marketplaces by analyzing seller behavior instead of just price.

SellerSense discovers undervalued listings using heuristics such as:

- Seller urgency
- Relocation signals
- Pricing anomalies
- Listing quality
- Trust indicators
- Market statistics

Instead of asking:

> "What is this item worth?"

SellerSense asks:

> "Why is this person selling?"
